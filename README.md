## Installation guide

# Laravel Backend

1. `composer install`
1. Rename or copy `.env.example` file to `.env`
1. Set your `(TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_NUMBER, TWILIO_FROM)` in your `.env` file. You can get an API key [here](https://www.twilio.com).
1. `php artisan key:generate`
1. `php artisan serve`
1. `php artisan websockets:serve` To Run Websocket Server For liveEvents.

# Vue Js Frontend

1. `npm install`
1. `npm run dev`
